# Installation

To install just clone the repo in the directory you want

```
$ git clone git@github.com:DevTommyy/rsm_front.git
```

then 'cd' into that directory

```
$ cd path/to/the/dir
```

finally execute the script to install it

```
# make the script executable
$ chmod +x install.sh
# run it
$ ./install.sh

```
